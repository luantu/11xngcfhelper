关于如何安装：
1. 打开chrome
2. 打开地址 chrome://extensions/
3. 将编译农场.crx拖动到浏览器界面内

关于无法启动：
详见解决31以上版本的Stable Beta，非商店扩展被强制停用的问题.doc

关于组策略添加模板完卡死问题
出现卡顿即可关闭组策略了，实际上已经加载好了

注意：
1. 仅支持记住编译工程、编译分支、编译产品、版本号
2. 编译类型不支持记住，默认完整编译
3. 不支持记住编译服务器
4. 仅支持chrome浏览器，仅在 41.0.2272.89（32bit）下测试ok，其余自行测试

ps. 代码在 https://github.com/luantu/11xngcfhelper
